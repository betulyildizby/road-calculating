 <h1>    About the Project   </h1>


<div align="center"> 
This project aims to provide the shortest road to the user according to the given city distance. By entering and determining 2 of the cities indicated on the map from the console, it shows the shortest road and from which cities to go, thanks to UCS, in the best and shortest way.
</div>

<div align="center"> 
Before you begin, make sure you have imported the csv file correctly. Check for compliance with UTF-8. Make sure the csv is called with the path specified in the code.

<h1>     After Running Code   </h1>
<div align="center"> 
Make sure the map is open in the browser. When you move the cursor over the map on the browser page that opens, you should see the country name and color. That's how you can be sure that the code works without errors.
